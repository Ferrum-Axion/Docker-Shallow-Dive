## Docker practices

## 117
Purpose: Practice basic cleanup, custom networking, and container connectivity. It teaches how network isolation works, how to attach running containers to another network, and how to validate connectivity between containers.

## 139
Purpose: Practice Docker volumes for persistent storage. It teaches that data in a named volume survives container removal and can be reused by a new container.

## 155
Purpose: Practice building a custom image and using build arguments and environment variables. It teaches how to configure a container at build-time and runtime and run a service in the foreground.
