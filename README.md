# SSH Server Containerization

## Overview
This project demonstrates how to containerize a legacy SSH server using Docker and Docker Compose.

The goal was to understand how SSH works, run it manually, and then migrate it into a containerized environment.

---

##  Features
- SSH server running inside a Docker container
- Connection from host machine using SSH client
- Supports both standalone Docker and Docker Compose
- Volume mounting between host and container
- Automated setup using script

---

##  Technologies Used
- Linux (Ubuntu)
- Docker
- Docker Compose
- OpenSSH Server
- Bash scripting
